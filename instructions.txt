Instructions
1. Install xampp
2. Place the project folder inside htdocs
3. Run xampp and start apache and mysql
4. In your browser go to localhost then go to phpmyadmin page
5. Create a new database, name it "laptop_rental_db"
6. Import sql file found at database folder of this project
7. In your browser go to localhost/laptoprental
8. If you renamed the project folder you should use that
    for example: localhost/nameoffolder
